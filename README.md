# Dynamic Content Management System (CMS)

A frontend-based Dynamic Content Management System built using HTML, CSS, and JavaScript.
This project allows users to add, edit, and delete content dynamically without page reload.

## Features
- Add, edit, and delete content
- Dynamic UI updates using JavaScript
- LocalStorage for data persistence
- Responsive admin-style dashboard
- Card-based layout with hover effects

## Technologies Used
- HTML
- CSS
- JavaScript
- Browser LocalStorage

## How to Run
1. Download or clone the repository
2. Open `index.html` in any web browser
